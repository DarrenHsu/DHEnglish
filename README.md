這是一個 swift 語言的 App，透過這個 App 我可以在上面：

1，新增單字及片語功能。

2，新增單字的例句功能。

3，修改並刪除新增的單字及例句。

4，翻頁瀏覽所有的單字、片語、例句等資料。

在此 App 中使用了 swift 連接 objectivect-c  的技巧，
並透過第三方元件 MagicRecord 來建置此 App。
